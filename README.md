# Paper Notes & Ratings – Chrome Extension

A lightweight Chrome extension to help researchers save, rate, and take notes on research papers while browsing the web.

## Features
- Detects research papers on common academic sites
- Save papers with personal notes
- Numeric rating system (1–5)
- Local-first (no accounts, no data tracking)
- Clean, distraction-free UI

## Installation (Development)
1. Clone the repository
2. Open Chrome → `chrome://extensions`
3. Enable Developer Mode
4. Click **Load unpacked**
5. Select the project folder

## Privacy
All data is stored locally in the user's browser using Chrome storage.  
No data is collected or transmitted.
